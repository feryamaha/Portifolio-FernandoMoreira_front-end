Autodata.F - Plataforma de Cursos Gratuitos

Olá! Eu sou a Autodata.F, uma plataforma de cursos criado por Fernando Moreira.
Eu organizo cursos gratuitos do YouTube para ele aprender programação rapidinho de forma gratuita e de qualidade.

O que é a Autodata.F?

Sou uma ferramenta pessoal que foi criada por Fernando Moreira para estudar de forma organizada usando cursos gratuitos do YouTube.
Meu objetivo é ajudar o Fernando a manter o foco nos estudos, evitando distrações com outros vídeos dentro do proprio Youtube, separando o tempo de aprendizado do tempo de descontração.
"Não sou uma plataforma aberta nem tenho fins lucrativos" — sou 100% de uso didático e pessoal do Fernando e todos os links e créditos dos cursos que eu compila aqui para ele estudar pertencem aos criadores e ao YouTube.
Veja mais em nossa Política de Proteção de Dados.

Quais tecnologias eu fui projetado?

  • Front-end: React e CSS Modules
  • Backend: Node.js com Express, PostgreSQL (Neon), bcrypt, CORS.
  • Infra: Vite, Vercel, Variáveis de Ambiente.

Minhas Interações e funcionalidades?

  • Formulario de gerenciamento: Acessando com senha permite Adicionar, editar e excluir cursos por categoria.
  • Navegação: Cards responsivos com menu fixo.
  • Vídeos: Player integrado com API React-YouTube.
  • Segurança: Acesso ao gerenciador de conteúdo protegido por senha criptografada.
  • Temas: Troca automática de temas com useTheme.
  • Voz (em breve): IA "Yaminuelle" integrada por comando de voz.

Política de Proteção de Dados

O Autodata.F é uma plataforma pessoal criada por Fernando Moreira com dois objetivos principais:

  1. Praticar Programação: Meu objetivo inicial foi servir como um projeto prático para Fernando Moreira.
  2. Estudo Organizado: Como Fernando é autodidata, ele decidiu usar o Autodata.F para estudar cursos gratuitos do YouTube.
  • Créditos: Todos os links e conteúdos dos cursos pertencem aos criadores e ao YouTube.
  • Uso Pessoal: Esta plataforma é de uso exclusivo de Fernando Moreira.
  • Proteção de Dados (LGPD): Não coletamos dados pessoais dos usuários, exceto para login, protegidos com bcrypt.
